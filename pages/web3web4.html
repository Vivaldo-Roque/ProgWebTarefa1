Web 3 e Web 4

Web 3 - Web semântica

O objetivo da Web Semântica é tornar os dados da Internet legíveis por máquina e não apenas por seres humanos.

  1. É a terceira geração da web, é aberta, inteligente, com tecnologias web semânticas, fontes de dados distribuídas, processamento de linguagem natural, aprendizado de máquina, máquina processamento.

  2. Os computadores podem interpretar informações como humanos e gerar distribuir conteúdo útil adaptado às necessidades dos usuários.

  3. Isso inclui transformar a web em um banco de dados, ummovimento para fazer conteúdo acessível por vários aplicativos não-navegador, a alavancagem de tecnologias de inteligência artificial.

  4. Tecnologias- RDF (Resource Description Framework), XML (eXtensible Markup Language), URI (Uniform Resource Identifier), API (Application Programming Interface), OAuth (Open Authorization), OWL (Web Ontology Language).

Web 4 - Web inteligente

Trata-se de deixar a web mais inteligente utilizando recursos da inteligencia artificial.

  1. É aprendizagem inteligente.

  2. Auto-aprendizagem.

  3. Auto-organização.

  4. Conecta todos os dispositivos no mundo real e virtual em tempo real.
