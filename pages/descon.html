Os elementos e atributos descontinuados no HTML5

Tags obsoletas

Os seguintes elementos não estão mais disponíveis em HTML5 e sua função é melhor manipulada por CSS.

Tags (Elements) Descrição
<acronym> Define um acrônimo
<applet> Define um applet
<basefont> Define uma fonte base para a página
<big> Define texto grande
<center> Define texto centralizado
<dir> Define uma lista de diretórios
<font> Define a fonte, tamanho e cor do texto
<frame> Define um quadro
<frameset> Define um conjunto de quadros
<isindex> Define um campo de entrada de linha única
<noframes> Define uma seção noframe
<s> Define o texto tachado
<strike> Define o texto tachado
<tt> Define o texto do teletipo
<u> Define o texto sublinhado

Atributos obsoletos

O HTML5 não possui nenhum dos atributos de apresentação que estavam no HTML4, pois suas funções são melhor tratadas pelo CSS. Alguns atributos do HTML4 não são mais permitidos no HTML5 e foram removidos completamente.

A seguir está a tabela com os atributos removidos e suas tags (elementos) impactadas correspondentes, ou seja. elementos dos quais esses atributos foram removidos permanentemente.
